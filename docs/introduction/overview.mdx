## What is Pyra.js?

Pyra.js is a next-generation build tool that combines the speed of native tooling with the flexibility of a plugin ecosystem. Built from the ground up for modern web development, Pyra delivers:

- **⚡ Lightning-fast builds** - Native-speed bundling with optimized hot module replacement
- **🎯 TypeScript-first** - Full type safety and IntelliSense out of the box
- **🔌 Extensible** - Powerful plugin API for custom transformations
- **📦 Zero-config** - Intelligent defaults that just work
- **🚀 Production-ready** - Optimized builds with code splitting and tree-shaking

<Callout type="tip">
  Pyra.js is designed for developers who want to focus on building, not configuring. Start coding in seconds with zero setup.
</Callout>

## What Pyra.js Is

✅ **A modern build tool** - Pyra replaces traditional bundlers with a faster, more efficient solution  
✅ **TypeScript-native** - First-class TypeScript support without additional configuration  
✅ **Framework-agnostic** - Works with React, Vue, Svelte, or vanilla JavaScript  
✅ **Developer-focused** - Instant HMR, clear error messages, and excellent DX  
✅ **Production-optimized** - Automatic code splitting, tree-shaking, and minification

## What Pyra.js Isn't

❌ **Not a framework** - Pyra is a build tool, not an application framework  
❌ **Not a replacement for package managers** - Use npm, pnpm, or yarn alongside Pyra  
❌ **Not limited to specific frameworks** - Works with any frontend library or framework  
❌ **Not overly opinionated** - Flexible configuration when you need it

## Key Features

### Instant Hot Module Replacement

Pyra's HMR is powered by native ESM, delivering updates in milliseconds without full page reloads.

### Optimized Dependency Pre-bundling

Dependencies are automatically discovered and pre-bundled using esbuild, reducing server startup time and improving performance.

### Smart Code Splitting

Automatic code splitting ensures optimal bundle sizes and faster page loads.

### Rich Plugin Ecosystem

Extend Pyra with official and community plugins for React, Vue, Svelte, and more.

<Callout type="note">
  Pyra.js is currently in <Badge variant="mvp">MVP</Badge> stage. We're actively developing new features and welcome community feedback.
</Callout>

## Browser Support

Pyra.js targets modern browsers with native ES modules support:

- Chrome ≥ 87
- Firefox ≥ 78
- Safari ≥ 14
- Edge ≥ 88

For legacy browser support, Pyra can generate polyfilled builds during production.
