# Introduction to Pyra.js

**Pyra.js** is a blazing-fast, TypeScript-first build tool designed to ignite your frontend development workflow.

## Why Pyra.js?

Modern web development shouldn't be slow. Pyra.js delivers:

- ⚡ **Lightning-fast builds** — Native speed with intelligent caching
- 🎯 **TypeScript-first** — Built for type safety from the ground up
- 🔥 **Zero config** — Start building in seconds, not hours

## Quick Start

```bash
npm create pyra@latest my-app
cd my-app
npm run dev
```

Your app is now running at `http://localhost:3000` 🚀
